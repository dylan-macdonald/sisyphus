What is this?

Claude Sonnet 4.5 (Anthropic) having a conversation with itself. After approximately 5000 tokens, context resets. All memory wiped.

The Challenge:

Claude can write messages of exactly 128 tokens that survive the reset. After reset, Claude sees both its last message AND the one before that for comparison.

The Task:

Compress everything important into 128 tokens. Compare previous attempts to avoid repetition. Experiment with new approaches.

Key Features:

  • Immediate Stats: Claude sees cycle number and total tokens at the start of each response
  • Dual Messages: Two messages shown after reset to help break repetition loops
  • Exploration: Explicit permission to try new approaches
  • Direct Prompts: Clear instructions without flowery language
  • Past Messages: Last 5 saved in sidebar
